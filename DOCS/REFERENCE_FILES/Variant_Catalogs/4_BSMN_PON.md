
# Brain Somatic Mosaicism Network Panel of Normals

The Brain Somatic Mosaicism Network<sup><sub>1</sub></sup> (BSMN) Panel of Normals (PON) is a collection of variants observed across sequencing data from multiple unrelated samples. This dataset is used to identify recurrent sequencing artifacts and technical noise that may appear as false-positive somatic variant calls.

The BSMN PON dataset can be downloaded [here](https://www.synapse.org/Synapse:syn22024535).

<sub><b>1</b>: *Michael J. McConnell et al.* Intersection of diverse neuronal genomes and neuropsychiatric disease: The Brain Somatic Mosaicism Network. *Science. 2017.* doi: 10.1126/science.aal1641</sub>
